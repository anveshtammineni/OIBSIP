# OIBSIP Web Development Task 3

## Project Title
Temperature Converter Web Application

## Objective
Develop an interactive temperature converter that supports Celsius, Fahrenheit, and Kelvin with input validation.

## Files
- index.html: Converter UI and JavaScript logic

## Features
- Convert between Celsius, Fahrenheit, and Kelvin
- Real-time form interaction with clear result output
- Input validation with helpful error messages
- Reset button to restore default values
- Responsive single-page interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run
1. Open the folder in VS Code.
2. Launch index.html in a browser.
3. Enter a temperature value.
4. Select source and target units.
5. Click Convert.

## Sample Test Cases
- Input: 100 C, Output: 212 F
- Input: 32 F, Output: 0 C
- Input: 273.15 K, Output: 0 C
